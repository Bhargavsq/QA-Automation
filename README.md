# QA Automation

A simple Selenium automation project built using Java, TestNG, and Maven to automate the login functionality of the Bung web application.

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven

## Test Cases

- ✅ Valid Login
- ✅ Invalid Username
- ✅ Invalid Password
- ✅ Empty Username
- ✅ Empty Password
- ✅ Empty Username & Password
